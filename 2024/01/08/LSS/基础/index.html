装箱与拆箱

装箱调用valueOf() 拆箱调用xxxvalue()；

public static void main(String[] args) {
        Integer i1 = 100;
        Integer i2 = 100;
        Integer i3 = 200;
        Integer i4 = 200;

        System.out.println(i1==i2);
        System.out.println(i3==i4);

    }
//Integer valueOf方法源码
public static Integer valueOf(int i) {
        if(i >= -128 && i <= IntegerCache.high)
            return IntegerCache.cache[i + 128];
        else
            return new Integer(i);
    }

i1 == i2输出true; i3 == i4输出为false

解析：因为i1与i2尚未出界，分析源码可知直接从IntegerCache缓存中取出对象的引用。

i3与i4出界，分别创建不同的对象

浮点数（Double Float）的实现方法又与上述有所不同

Integer i = new Integer(xxx) 不会自动装箱

Integer i =xxx; 会自动装箱

具体到牛客网本题：

        Integer i01=59; //返回的是IntegerCache中的对象的引用
        int i02=59; //同上
        Integer i03=Integer.valueOf(59); //装箱 同上
        Integer i04=new Integer(59); //new  堆中新开辟了内存地址

JVM中，new 一个对象存储在其堆中，而调用数据是通过栈。对象本身较消耗资源。而基本数据类型的变量不需要经过new创建，直接在栈中存储高效。

与此同时，基本数据类型并非是面向对象的，如利用集合类时开发者无法将int double等类型放入。为了基本类型也具有对象的特征，就出现了包装类。

Integer i =10; //自动装箱 int b= i; //自动拆箱

int的自动装箱都是通过Integer.valueOf()方法来实现的，Integer的自动拆箱都是通过integer.intValue

I/O与文件

File类是对文件整体或者文件属性操作的类，例如创建文件、删除文件、查看文件是否存在等功能，不能操作文件内容；文件内容是用IO流操作的。

Java中对于文本文件和二进制文件，都可以当作二进制文件进行操作。

default不能修饰变量 abstract一般修饰方法和类

Java将数据的输入输出抽象为流

字节 字符

InputStream/OutputStream 字节 二进制

Reader/Writer 字符或字符串 图音视频建议用字节流

异常

根据官方的JVM规范： 如果try语句里有return，返回的是try语句块中变量值。 详细执行过程如下：

  1. 如果有返回值，就把返回值保存到局部变量中；
  2. 执行jsr指令跳到finally语句里执行；
  3. 执行完finally语句后，返回之前保存在局部变量表里的值。

如果try，finally语句里均有return，忽略try的return，而使用finally的return.

public class Test { 
    public static void main(String[] args) { 
        System.out.println("return value of getValue(): " +
        getValue()); 
    } 
     public static int getValue() { 
         try { 
             return 0; 
         } finally { 
             return 1; 
         } 
     } 
 }

public class NameList
{
    private List names = new ArrayList();
    public synchronized void add(String name)
    {
        names.add(name);
    }
    public synchronized void printAll()     {
        for (int i = 0; i < names.size(); i++)
        {
            System.out.print(names.get(i) + ””);
        }
    }

    public static void main(String[]args)
    {
        final NameList sl = new NameList();
        for (int i = 0; i < 2; i++)
        {
            new Thread()
            {
                public void run()
                {
                    sl.add(“A”);
                    sl.add(“B”);
                    sl.add(“C”);
                    sl.printAll();
                }
            } .start();
        }
    }
}
